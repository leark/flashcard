JS & jQuery Flashcards

JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A character or characters that determine the action that is to be performed or considered.

Variables

A value that can change, depending on conditions or on information passed to the program.

Variable Naming Conventions
  • Snakecase: Words are delimited by an underscore.
  • Pascalcase: Words are delimited by capital letters.
  • Camelcase: Words are delimited by capital letters, except the initial word.
Functions

"Self contained" modules of code that accomplish a specific task.

Methods

A programmed procedure that is defined as part of a class and included in any object of that class.

Arguments

A value that is passed between programs, subroutines or functions.

Parameters

A special kind of variable in computer programming language that is used to pass information between functions or procedures.

Return

A statement that instructs a program to leave the subroutine and go back to the return address.

Chaining methods

A common syntax for invoking multiple method calls in object-oriented programming languages.

Strings

Used for data values that are made up of ordered sequences of characters

Booleans

A value that is either true or false.

Undefined

Anything that lacks a description or is not declared.

NaN

Not A Number.

Null

The value null represents the intentional absence of any object value.

Alert

Alert instructs the browser to display a dialog with an optional message, and to wait until the user dismisses the dialog.

Comments

Comment is used to add explanatory notes to the code or to prevent the browser from interpreting specific parts

jQuery

jQuery is a JavaScript Library that focuses on simplifying DOM manipulation, AJAX calls, and Event handling.

Attributes

HTML attributes are special words used inside the opening tag to control the element's behaviour. For example, "href" in <a> tag.
